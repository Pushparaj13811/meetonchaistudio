## Naming Philosophy

Great software studio names should:
1. **Be memorable** (short, unique, pronounceable)
2. **Scale globally** (work across cultures)
3. **Not limit services** (avoid "MobileApps4U" type names)
4. **Have available .com domain**
5. **Be defensible** (can trademark)
6. **Tell a story** (or at least not require explanation)
